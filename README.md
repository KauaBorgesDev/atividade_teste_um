# Sistema Simples de Login

## Objetivo

Sistema desenvolvido em PHP para realizar login, cadastro e listagem de usuários utilizando banco de dados MySQL.

## Tecnologias Utilizadas

* PHP
* MySQL
* HTML
* CSS
* GitHub

## Estrutura do Projeto

* index.php = Tela de login.
* home.php = Página principal.
* logout.php = Encerrar sessão.
* connect.php = Conexão com banco.
* navbar.php = Cabeçalho.
* table.php = Tabela de usuários.
* style.css = Estilos.
* banco.sql = Banco de dados.

## Funcionamento

O usuário faz login com usuário e senha. Após entrar no sistema, pode cadastrar novos usuários e visualizar os usuários já cadastrados. Ao clicar em sair, a sessão é encerrada.

* Manipulação de sessões.
* Formulários HTML.
* Consultas SQL.
* Organização básica de projetos web.
